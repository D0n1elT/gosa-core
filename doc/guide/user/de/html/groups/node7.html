    • Verwalten: Um die Kiosk-Profile zu verwalten, klicken Sie auf den Knopf Verwalten. Hier können Sie vorhandene Kiosk-Profile entfernen, sowie neue Kiosk-Profile hinzufügen.
  • Auflösung änderbar während des Betriebs: Wenn Sie möchten, dass Mitglieder dieser Gruppe die Auflösung Ihres Arbeitsplatzrechners während der Sitzung ändern dürfen, aktivieren Sie diese Option.
  • Auflösung: Legen Sie die Auflösung für Arbeitsplatzrechner, die dieses Kiosk-Profil verwenden, fest, indem Sie die gewünschte Auflösung aus der Liste wählen. Um automatisch die beste Auflösung zu verwenden, wählen Sie auto.
