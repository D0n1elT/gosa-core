Alternative Adressen

Alternative Adressen sind weitere Adressen, unter denen der gemeinsam genutzte Ordner Mails empfangen soll.

Verwenden Sie den Knopf Hinzufügen um eine alternative Adresse hinzuzufügen. Wenn Sie eine Adresse entfernen möchten, markieren Sie diese und drücken Sie auf den Knopf Entfernen.
