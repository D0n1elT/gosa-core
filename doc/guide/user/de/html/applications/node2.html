Eine Anwendung hinzufügen / Eine bestehende Anwendung bearbeiten

Um eine Anwendung hinzuzufügen, klicken Sie auf den Knopf (Er befindet sich oberhalb der Liste). Wenn Sie keine neue Anwendung hinzufügen wollen, sondern eine vorhandene bearbeiten möchten, klicken Sie entweder auf den Eintrag in der Liste (z.B. gimp - [Bildbearbeitung]) oder auf den Knopf des entsprechenden Eintrags.

Es öffnet sich eine Seite mit drei Reitern (Allgemein / Optionen / Referenzen). Diese dienen dazu, die Eigenschaften der Anwendung logisch aufzuteilen (Unter Allgemein werden die generellen Eigenschaften der Anwendung konfiguriert, Optionen dient dazu, Umgebungsvariablen für die Anwendung zu setzen und Referenzen zeigt an, von welchen Objekten (Benutzer, Gruppen, etc.) die gewählte Anwendung verwendet wird).

Um Änderungen zu speichern, verwenden Sie den Knopf Anwenden. Um die vorgenommenen Änderungen zu verwerfen und wieder in die Liste der Anwendungen zu gelangen, verwenden Sie den Knopf Abbrechen.

Die Felder, die mit einem roten Sternchen enden, sind Pflichtfelder und müssen zwingend ausgefüllt werden (GOsa wird die Änderungen sonst nicht übernehmen).

Im Folgenden werden die Reiter ausführlich beschrieben.
