Gruppenmitgliedschaft

Die Liste zeigt alle UNIX-Gruppen, in denen der Benutzer Mitglied ist. Um eine weitere Mitgliedschaft hinzuzufügen, klicken Sie auf den Knopf Hinzufügen unterhalb der Liste. Markieren Sie eine oder mehrere Gruppen und klicken Sie auf hinzufügen. Um den Benutzer aus einer oder mehrerer Gruppen zu entfernen, markieren Sie eine oder mehrere Gruppen und klicken auf Entfernen unterhalb der Liste.
