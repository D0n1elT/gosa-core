Telefonnummern

In dieser Liste werden die Telefonnummern aufgeführt, die dem Benutzer zugeordnet sind.

  • Nummer hinzufügen: Geben Sie die Nummer in die Eingabezeile ein und drücken Sie den Knopf Hinzufügen
  • Nummer entfernen: Markieren Sie die Nummer aus der Liste und drücken Sie den Knopf Entfernen.
