System-Vertrauen

Dieser Abschnitt dient der Definition von Zugriffsbeschränkungen auf definierte Systeme, Geräte, etc.

Die möglichen Vertrauensmodi:

  • deaktiviert: Der Benutzer darf sich auf allen Hosts anmelden.
  • Vollzugriff: Der Benutzer darf sich auf allen Hosts anmelden.
  • erlaube Zugriff auf diese Hosts: Der Benutzer darf sich nur auf den aufgeführten Hosts anmelden.

    • Hinzufügen: Um einen oder mehrere Hosts in die Liste aufzunehmen, klicken Sie auf den Knopf Hinzufügen. Aus der Liste der Hosts wählen Sie einen oder mehrere zulässige Hosts und klicken Sie auf den Knopf Hinzufügen. Die Hosts werden nun in der Liste aufgeführt. Um den Vorgang abzubrechen, klicken Sie auf den Knopf Abbrechen.
    • Entfernen: Um einen oder mehrere Hosts aus der Liste zu entfernen, markieren Sie den oder die gewünschten Hosts aus der Liste und drücken Sie den Knopf Entfernen. Die Hosts werden nun nicht mehr in der Liste aufgeführt.
