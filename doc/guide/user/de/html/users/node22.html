Alternative Adressen

Alternative Adressen sind weitere Adressen, unter denen der Benutzer Mails empfangen soll.

Verwenden Sie den Knopf Hinzufügen um eine alternative Adresse hinzuzufügen. Wenn Sie eine Adresse entfernen möchten, markieren Sie diese und klicken auf den Knopf Entfernen.
