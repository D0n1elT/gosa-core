Databases

Ce point regroupe toutes les bases de données qui peuvent être associés au serveur. L'administrateur peut cocher une ou plusieurs propositions de bases de données selon la configuration du système et les droits d'accès des utilisateurs.

L'administrateur à le choix entre :

  • Kerberos kadmin access

    Kerberos Realm  
    Admin user  
    Mot de passe  


  • FAX database

    FAX DB user  
    Mot de passe  


  • Asterisk Management

    Asterisk DB user  
    Mot de passe  
    Country dial prefix  
    Local dial prefix  


  • IMAP admin access

    Server identifier  
    Connect URL  
    Admin user  
    Mot de passe  
    Sieve port  


  • Logging database

    Logging DB user  
    Mot de passe
