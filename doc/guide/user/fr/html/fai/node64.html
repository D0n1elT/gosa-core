Liste des modèles

Le champ "Liste des modèles" contient les modèles sur lesquelles trois actions peuvent se faire :

- pour le téléchargement du modèle

- pour éditer les attributs du modèle

- pour supprimer le modèle

Sélectionnez un modèle dans la liste existante et cliquez sur le bouton 'Editer' pour accéder aux informations sur ce modèle.

Cliquez sur le bouton 'Appliquer' pour sauver les modifications apportées au modèle ou sur le bouton 'Annuler' si aucune modification n'a été apportée au modèle. Vous voilà à nouveau sur le champ "Liste des modèles".

Cliquez ensuite sur le bouton 'Enregistrer' pour sauver l'ensemble des attributs du modèle et revenir sur la page principale.
