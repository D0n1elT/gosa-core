Paquets utilisés

Le champ "Paquets utilisés" reprend la liste des paquets inclus dans la classe.

Pour supprimer un ou plusieurs paquets, sélectionnez dans la liste les paquets à effacer, ensuite cliquez sur le bouton 'Supprimer'.

Le bouton 'Configurer' sert à configurer un paquet debian à travers l'interface de gestion GOsa . Sélectionnez dans la liste le paquet à configurer, cliquez sur le bouton 'Configurer' :

- Soit le paquet est configuré par défaut et n'a pas d'options debconf,

- Soit le paquet a des options debconf par défaut, vous pouvez donc changer la configuration classique du paquet. Cliquez ensuite sur le bouton 'Appliquer' pour sauver la nouvelle configuration du paquet et revenir sur la liste des "Paquets utilisés". Cliquez enfin sur le bouton 'Enregistrer' pour sauvegarder l'ensemble des attributs du paquet et revenir sur la page principale.

Le bouton 'Activer/désactiver le marqueur à effacer' sert à marquer les paquets que vous désirez effacer :

- Sélectionnez un paquet,

- Cliquez sur 'Activer/désactiver le marqueur à effacer'. La paquet est marqué dans la liste comme élément à enlever,

- Pour l'activer il faut à nouveau sélectionner le paquet et cliquer sur 'Activer/désactiver le marqueur à effacer'.

- Cliquez enfin sur le bouton 'Enregistrer' pour sauver l'ensemble des attributs de liste de paquets ou sur le bouton 'Annuler' si aucune modification n'a été apportée. Vous retournez sur la page principale.
