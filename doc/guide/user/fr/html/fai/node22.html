Création d'une classe de scripts dans GOsa ( pour créer la classe)

Entrez le nom de la classe FAI manuellement Si vous choisissez de créer une nouvelle classe de scripts, sélectionnez la première option et insérez le nom de la nouvelle classe.
Choisissez le nom de la classe FAI parmi les classes existantes Si vous choisissez d'inclure la nouvelle classe dans une classe existante, sélectionnez la deuxième option et sélectionnez dans le menu déroulant le nom de la classe existante


Ensuite cliquez sur le bouton 'Continuer'.
