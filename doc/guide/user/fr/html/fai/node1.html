FAI

Cliquez sur le bouton FAI à gauche dans la partie Administration . La page Installation complète en mode automatique s'affiche. La page est constituée de plusieurs tableaux :

LE TABLEAU PRINCIPAL est destiné à la création et à la gestion des installations en mode automatique.

Les icônes en entête ( , , ) servent à modifier l'affichage selon le département, elles prédominent sur toutes autres formes de sélection d'affichage. Ces icônes sont reliées au champ "Base".

Le champ "Base" sert à afficher les différents départements existant dans l'organisation.

Pour créer une classe, l'administrateur doit cliquer selon son choix sur un des boutons suivant:

  • pour créer des profils,
  • pour créer des tables des partitions,
  • pour créer des scripts,
  • pour créer des connexions,
  • pour créer des variables,
  • pour créer des modèles,
  • pour créer des listes des paquets.

L'icône sert à copier/coller des classes dans la liste. La fonction est utilisée conjointement avec l'icône dans la colonne intitulée "Actions" :

1. choisissez une classe et sélectionnez la grâce à l'icône ,

2. l'icône s'active,

3. cliquez sur l'icône activée , elle renvoie à l'assistant pour le copier/coller,

4. choisissez un nouveau nom pour votre classe,

5. cliquez sur le bouton 'Enregistrer',

6. la nouvelle classe apparaît dans la liste.

LE DEUXIÈME TABLEAU dans la partie supérieure à droite est le cadre " information", il apporte des informations sur ce qu'il est possible de faire sur la page courante. Le contenu du tableau change en fonction de la page affichée.

LE TROISIÈME TABLEAU dans la partie médiane à droite reprend dans un menu déroulant les différentes versions de la distribution Debian GNU/Linux pour lesquelles une configuration pour FAI existe.

Il est aussi possible à partir de ce tableau d'indiquer que l'on désire créer une nouvelle branche ou de figer une configuration existante.

LE QUATRIÈME TABLEAU dans la partie inférieure droite sert à modifier selon la préférence l'affichage des données dans le tableau principal. Le type de recherche s'adapte en fonction du contenu du tableau principal.

Ce tableau intitulé "Filtres " permet d'effectuer une recherche :

- en cliquant sur l'astérisque (étoile : *) pour voir apparaître toutes les classes FAI;

- en cliquant sur une lettre et toutes les classes dont le nom débute par cette lettre s'afficheront;

- en cliquant sur un numéro et toutes les classes dont le nom débute par ce numéro s'afficheront;

- en cochant une ou plusieurs proposition(s) pour une recherche d'ensemble;

- ou encore la recherche rapide : remplissez le champ par le nom d'une classe et ensuite cliquez sur le bouton 'Appliquer le filtre'.

Le tableau principal se décline en quatre colonnes :

  • La première colonne contient les icônes indiquant de manière graphique le type de la classe. Les icônes utilisées sont :

    . pour désigner les classes profils,

    . pour désigner les classes de table de partitions,

    . pour désigner les classes de scripts,

    . pour désigner les classes de connexions,

    . pour désigner les classes de variables,

    . pour désigner les classes de modèles,

    . pour désigner les classes de liste de paquets

  • La deuxième colonne contient les noms de toutes les classes existantes dans cette configuration,
  • La troisième colonne indique le type de la classe,
  • La quatrième colonne contient les actions que l'on peut exécuter sur les classes :

    - l'icône ? , ,

    - l'icône est utilisée conjointement avec l'icône pour faire du copier/coller ( voir plus haut),

    - l'icône est utilisée pour éditer les informations de la classe sélectionnée,

    - l'icône est utilisée pour supprimer la classe sélectionnée.
