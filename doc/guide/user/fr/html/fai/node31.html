Liste des scripts

Le champ "Liste des scripts" contient les scripts sur lesquelles trois actions peuvent se faire :

- pour le téléchargement du script

- pour éditer le code du script

- pour supprimer le script

Un fois édité, le code du script peut être modifié ou remplacé.

Cliquez sur le bouton 'Appliquer' pour sauver les modifications apportées au script ou sur le bouton 'Annuler' si aucune modification n'a été apportée au script. Vous voilà à nouveau sur le champ "Liste des scripts".

Cliquez ensuite sur le bouton 'Enregistrer' pour sauver l'ensemble des attributs du script et revenir sur la page principale.
