Liste des noms de variables utilisées

Sélectionnez une variable qui se trouve dans le champ "Liste des noms de variables utilisées", cliquez sur le bouton 'Editer' pour accéder aux informations sur cette variable ou cliquez sur le bouton 'Supprimer' pour effacer la variable.

Cliquez ensuite sur le bouton 'Enregistrer' pour sauver les modifications apportées à la variable et revenir sur la page principale.
