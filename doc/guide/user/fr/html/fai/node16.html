Liste de partitions

Pour créer une nouvelle partition dans le disque dur cliquez sur le bouton 'Ajouter une partition' :

Type Sélectionnez dans la liste déroulante le type de la partition
FS type Sélectionnez dans la liste déroulante le type de système de fichier
Point de montage Indiquez le point de montage dans l'arborescence du disque
Taille en MB Indiquez la taille de la partition en MB
Options de montage Indiquez les options à passer au système de fichier lors du montage
Option du système de fichier Indiquez les options du système de fichier
Protéger Cochez si vous désirez protéger cette partition
Supprimer Cliquez sur le bouton 'Supprimer' pour effacer la partition


Cliquez sur le bouton 'Enregistrer' pour sauver la partition et revenir sur le champ "Disques".

Cliquez enfin sur le bouton 'Enregistrer' pour sauver l'ensemble du contenu de la nouvelle partition et revenir sur la page principale.
