Concept d'une classe profil

Si on doit considérer tout le processus d'installation en mode automatique, la classe profil rassemble l'ensemble des classes constituant un système à installer.

La classe profil regroupe l'ensemble des classes FAI (ou fichiers de configuration) choisies par l'administrateur pour l'installation et la configuration du système.
