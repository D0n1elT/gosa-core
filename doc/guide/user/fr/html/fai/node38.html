Attributs des connexions

Tâche Sélectionnez dans la liste déroulante la méthode avec laquelle le script sera éxecuté lors de l'installation du système


Pour ajouter un nouveau script dans le champ 'Script' :

- Cliquez sur le bouton 'Naviguer'.

- Vous sélectionnez un script dans votre arborescence.

- Faites un double clic sur le script sélectionné dans votre arborescence ou utilisez le bouton 'Ouvrir' pour afficher le chemin du script dans le champ 'Naviguer'.

- Ensuite cliquez sur le bouton 'Importez un script'. Le code du script se rajoutera automatiquement dans le champ 'Script'.

- Cliquez sur le bouton 'Appliquer' pour sauver le script et revenir sur le champ "Liste des scripts de connexions".

- Cliquez enfin sur le bouton 'Enregistrer' pour sauver l'ensemble des attributs du nouveau script et revenir sur la page principale.
