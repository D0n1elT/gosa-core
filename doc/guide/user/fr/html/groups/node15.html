Transférer le messages vers un membre n'appartenant pas au groupe

Des utilisateurs peuvent recevoir les messages d'un groupe sans pour autant en être membre. Il suffit d'insérez les adresses de ces utilisateurs dans le champ qui précède les boutons Ajouter, Ajouter en local et Supprimer. Le bouton Ajouter en local renvoie aux adresses déjà configurées dans GOsa.
