Applications

Pour activer l'extension Applications cliquez sur Créer des applications.

Les applications disponibles sont les applications existantes au sein de la société. L'administrateur dispose de la liste des applications dans le champ droit. Il peut attribuer une ou plusieurs applications à un groupe. Il suffit de cliquer sur le nom d'une application et d'utiliser les flèches pour faire voyager l'application d'un champ à l'autre. Pour terminé cliquer sur le bouton Editer les paramètres.
