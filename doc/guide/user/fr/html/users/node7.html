Appartient aux groupes

Utilisez les boutons Ajouter et Supprimer pour insérer ou effacer les groupes dans le champ vide. Le bouton Ajouter renvoie à la liste des groupes crées par l'administrateur. L'utilisateur peut appartenir à un ou plusieurs groupes.
