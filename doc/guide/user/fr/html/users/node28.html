Compte FTP

Si vous activez le compte FTP, vous pouvez régler la bande passante montante et descendante, le Ratio des données et le Quota à ne pas dépasser pour le transfert de fichiers . Vous pouvez aussi à partir de cet endroit désactiver le compte FTP.
