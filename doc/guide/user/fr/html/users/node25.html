Options d'accès

  • Tout comme dans la création du compte Unix l'administrateur a la possibilité de gérer la manière dont l'utilisateur dispose de son mot de passe. L'administrateur doit veiller à faire un choix cohérent pour éviter les conflits entre les différentes propositions.
  • Le champ à droite doit être rempli par les noms de stations à partir desquelles l'utilisateur peut se connecter au serveur. L'administrateur doit utiliser les boutons Ajouter et Supprimer pour y faire apparaître les noms de stations. Le bouton Ajouter renvoie à la liste des stations.
