Options d'accès

  • Il s'agit de la gestion du mot de passe de l'utilisateur. Différentes propositions peuvent être cochées à la fois mais l'administrateur doit veiller à rester cohérent pour éviter les conflits dans l'application de ses choix.
  • Le champ à droite doit être rempli par les noms de stations à partir desquelles l'utilisateur peut se connecter. L'administrateur doit utiliser les boutons Ajouter pour y insérer les noms de stations. Le bouton Ajouter renvoie à la liste des stations. Utilisez le bouton Supprimer pour effacer des stations.
