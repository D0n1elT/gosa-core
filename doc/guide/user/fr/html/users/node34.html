Méthodes de distribution

  • Desactiver temporairement l'utilisation du fax : interdit l'utilisation du fax par cet utilisateur.
  • Délivrer les fax comme des mails : envoie les fax sous le format de distribution défini dans la messagerie de l'utilisateur
  • Imprimer directement les fax : permet de sélectionner l'imprimante à laquelle envoyer les fax reçus par cet utilisateur
